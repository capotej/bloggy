subclass(base_name, &body)

Creates a new subclass of self, with a name "under" our own name. Example: 

  x = Foo::Bar.subclass('Zap'){}
  x.name # => Foo::Bar::Zap_1
  x.superclass.name # => Foo::Bar

[Validate]
